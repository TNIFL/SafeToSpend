SafeToSpend(쓸수있어)는 관련 법령에 따라 이용자의 개인정보를 처리합니다.

1. 수집 항목

  • 계정: 이메일, 비밀번호 해시, 계정 생성/플랜 상태 정보
  • 서비스 데이터: 거래내역, 세금/건보 입력값, 증빙/영수증 파일 및 메타데이터, 문의 내용
  • 결제/구독 데이터: 결제 시도/이벤트, 구독 상태 정보
  • 보안 데이터: 세션/CSRF 토큰, API 토큰 메타, IP·User-Agent(보안 목적)

2. 수집 방법

  • 회원가입·로그인·설정 화면 직접 입력
  • CSV/계좌연동/파일 업로드
  • 서비스 이용 중 자동 생성(분류/추정/로그)

3. 이용 목적

  • 이용자 인증 및 계정 관리
  • 거래·증빙 정리, 세금·건보 추정 결과 제공
  • 결제/구독 처리 및 권한 관리
  • 고객문의 처리, 보안 및 부정 이용 방지

4. 보유 및 파기

  • 회원 탈퇴 시 관련 데이터 삭제 절차를 진행합니다.
  • 증빙 파일은 기본 보관기한(기본 7년) 만료 시 정리 대상이 됩니다.
  • 탈퇴 시 물리 파일 삭제를 시도하며 일부 실패분은 후속 정리될 수 있습니다.

5. 제3자 제공/외부 서비스

  • 일반적인 마케팅 목적 제3자 제공 기능은 현재 코드 기준 확인되지 않았습니다.
  • 서비스 제공을 위해 결제(토스), 계좌연동(팝빌), 영수증 파싱(OpenAI) 연동을 사용합니다.

6. 이용자 권리

  • 정보 열람/수정, 비밀번호 변경, 계정 삭제 요청이 가능합니다.
  • 문의는 /support에서 접수할 수 있습니다.

7. 안전성 확보조치

  • 비밀번호 해시 저장, 세션 쿠키 보안 속성, CSRF 검증
  • API 토큰 검증 및 refresh token 회전/폐기
  • 보안 헤더 적용 및 민감 로그 필터링
