참고자료 / 추가설명 업로드
사용자가 직접 만든 엑셀, 메모 PDF, 설명 스크린샷 같은 자유형 자료를 공식자료/증빙과 분리해서 보관하는 채널입니다.
세무사 패키지 공식자료 업로드 증빙 보관함
이 채널의 용도 참고용
- 이 자료는 자동 반영되지 않습니다.
- 세무사 참고용 자료로만 보관됩니다.
- 공식자료/증빙과는 별도 관리됩니다.
- 이후 패키지의 참고자료/추가설명 영역으로 들어갈 예정입니다.
올릴 수 있는 파일 v1 범위
- PDF
- XLSX / CSV
- TXT
- JPG / PNG
사용자가 직접 만든 정리 엑셀, 추가 설명 문서, 전달 메모 파일을 받기 위한 범위만 열어 두었습니다.
이 채널에 올리면 좋은 자료 설명용
- 거래 설명 메모
- 경비 설명 자료
- 수입 구조 설명 자료
- 납부/자격 상태에 대한 보조 설명 자료
- 세무사에게 맥락 전달이 필요한 파일
거래만으로는 설명되지 않는 배경, 사용자가 직접 정리한 표, 세무사에게 같이 전달할 메모는 이 채널에 두는 편이 맞습니다.
