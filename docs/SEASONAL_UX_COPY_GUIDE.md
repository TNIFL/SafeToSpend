# Seasonal UX Copy Guide

## 카피 원칙

- 세법 이름보다 `지금 해야 할 일`을 먼저 말한다.
- 5월/11월의 제도 설명은 짧게 하고, 행동은 명확하게 쪼갠다.
- `잘 모르겠어요`를 허용하는 톤을 유지한다.
- 완료/진행 중/지금 확인 상태를 한눈에 보이게 한다.

## 시즌별 메인 카피

### 5월 신고 시즌

- 메인 문구: `이번 시즌은 작년 수입과 비용을 정리하는 달이에요`
- 보조 문구: `지금 3가지만 보면 신고 전에 숫자, 영수증, 전달 자료를 한 번에 정리할 수 있어요.`

### 11월 중간예납 시즌

- 메인 문구: `이번 시즌은 상반기 기준으로 미리 점검하는 시기예요`
- 보조 문구: `지금 부족한 것부터 정리하면 11월에 숫자가 갑자기 흔들리는 일을 줄일 수 있어요.`

### 비시즌

- 메인 문구: `이번 달 리듬에 맞춰 필요한 것만 정리해요`
- 보조 문구: `다음 시즌에 급해지지 않도록 지금 보이는 결과에서 필요한 것만 이어서 하면 돼요.`

## 생활 언어 치환

| 기존 표현 | 시즌 UX 우선 표현 |
| --- | --- |
| 종합소득세 신고 준비 | 작년 수입과 비용 정리 |
| 중간예납 점검 | 상반기 기준 미리 점검 |
| 기납부세액 확인 | 이미 빠진 세금 확인 |
| 필요경비 반영 | 일하면서 쓴 비용 반영 |
| 증빙 보완 | 영수증/메모 보강 |

## CTA 원칙

- `기본 정보 보완하기`
- `정리하기`
- `패키지 보기`
- `부족분 확인하기`

세무 용어가 필요한 경우에도 CTA 전면 문구로는 쓰지 않는다.

## 피해야 할 톤

- `지금 바로 신고 기준을 확정하세요`
- `중간예납 대상 여부를 판정합니다`
- `필수 세무 입력을 완료하지 않으면 안 됩니다`

## 권장 톤

- `지금 3가지만 보면 돼요`
- `지금 부족한 것부터 정리해요`
- `다음 시즌 전에 미리 맞춰 둘 수 있어요`
- `확인하면 더 정확해져요`
