# 해지/환불 정책(초안)

작성일: 2026-03-11
상태: 내부 검토용 초안

## 1. 기본 원칙
- 본 서비스는 현재 정책상 환불을 제공하지 않습니다.
- 해지는 즉시 종료가 아니라 기간 종료 해지(cancel at period end) 방식입니다.

## 2. 해지 정책
- 사용자가 해지를 요청하면 자동 갱신이 중단됩니다.
- 이미 결제된 기간 동안은 기존 플랜 권한을 유지합니다.
- 결제 주기 종료 시점(`cancel_effective_at`)에 해지가 완료됩니다.

## 3. 업그레이드 관련
- 업그레이드 시 전체 금액 즉시 청구
- 기존 결제분 환불 없음
- 업그레이드 성공 시 즉시 상위 플랜 적용

## 4. 추가 계좌(add-on) 관련
- 추가 계좌는 반복 과금 항목입니다.
- 중간 추가: 남은 기간 일할 계산 즉시 청구
- 다음 정기결제일부터 월 정상금액 합산 청구
- 추가 계좌 감소 요청: 다음 청구일부터 반영, 당월 환불 없음

## 5. 결제 실패 관련
- 결제 실패 시 3일 유예기간 적용
- 유예기간 동안 기존 유료 권한 유지
- 유예기간 경과 후 결제 실패 지속 시 past_due 전환

## 6. 문의
- 인앱 문의(`/support`)를 통해 해지/결제 문의 가능

## 7. 법률 검토 필요 사항
- 환불 불가 정책의 고지/동의 방식
- 결제 분쟁 대응 문구 및 예외 처리 범위
