# 민감정보 처리 정책 (서비스 기본)

## 1) 기본 원칙
- 서비스는 건보료/세금 추정과 증빙 정리를 위해 필요한 최소 정보만 처리합니다.
- 아래 민감 서류는 기본 흐름에서 수집/저장하지 않습니다.
  - 주민등록번호가 포함된 신분증 사본
  - 가족관계증명서/주민등록등본/초본
  - 여권/운전면허증 등 개인식별 서류

## 2) 업로드/저장 정책
- 사용자가 직접 업로드한 증빙만 저장합니다.
- 파일명에 민감 서류 패턴(예: `idcard`, `주민등록`, `가족관계`)이 있으면 업로드를 차단합니다.
- 증빙은 보관기간 정책에 따라 관리되며, 사용자는 언제든 삭제 요청/초기화를 할 수 있습니다.

## 3) 패키지/전달 정책
- 세무사 전달 패키지는 사용자가 생성할 때만 만들어집니다.
- 기본 패키지에는 서비스에 업로드된 증빙만 포함됩니다.
- 민감 인적서류가 정말 필요한 경우, 앱 업로드 대신 세무사와 합의한 별도 안전 채널로 전달해야 합니다.

## 4) 표시/로그 정책
- 사용자 화면에는 민감정보 원문 노출을 최소화합니다.
- 서버 로그에는 민감 본문(주소/계좌/개인식별)을 남기지 않습니다.
