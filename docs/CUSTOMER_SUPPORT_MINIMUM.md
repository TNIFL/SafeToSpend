# Customer Support Minimum

작성일: 2026-03-11
목적: 1인 운영 기준 최소 문의 대응 구조 확정

## 1) 공식 문의 채널(현재 확정)
- 인앱 문의: `/support`
- 사용자 문의 목록: `/support/my`
- 관리자 답변 경로: `/admin/inquiries`

근거 코드:
- `routes/web/support.py`
- `templates/support/*`

## 2) 사용자에게 보이는 공개 위치
- 현재: 로그인 사용자 상단 내비게이션 `문의` 메뉴(`templates/base.html`)
- 현재 공백: 비로그인/랜딩 페이지 하단 footer 고정 링크 미구현

## 3) 운영 현실 기준 응답 기대치(초안)
- 1차 응답 목표: 영업일 24시간 내
- 일반 해결 목표: 영업일 48~72시간
- 결제/데이터 손실 의심 건: 우선 처리

## 4) 최소 공지/안내 연계
- 결제 실패/지연/장애 시 공지 문구 템플릿은 `docs/STATUS_NOTICE_MINIMUM.md` 사용
- 결제/해지 정책 문의는 정책 문서 초안 링크와 함께 안내

## 5) 오픈 판정 관점
- 문의 채널 자체는 존재(부분 충족)
- 단, 비로그인 사용자 포함 공개 노출 위치(footer/help) 미구현이면 `오픈 전 필수 미완료`로 간주
