# 결제/구독/자동결제 안내(초안)

작성일: 2026-03-11
상태: 내부 검토용 초안

## 1. 플랜 및 월 요금
- basic: 월 6,900원
- pro: 월 12,900원
- 추가 계좌(add-on): 계좌 1개당 월 3,000원 (반복 과금)

## 2. 플랜별 핵심 권한
- free: 자동 계좌 연동/세무사 전달 패키지 다운로드 불가
- basic: 계좌 1개 자동 연동, 4시간 동기화, 패키지 다운로드 가능
- pro: 계좌 2개 자동 연동, 1시간 동기화, 패키지 다운로드 가능

## 3. 결제 흐름
- 사용자가 구독/업그레이드/추가계좌 CTA를 선택
- 결제수단 미등록 시 결제수단 등록 진행
- 등록 완료 후 결제 시도 생성
- 최종 확정은 reconcile 경로 검증 후 반영

## 4. 업그레이드 정책
- 전체 금액 즉시 청구
- 기존 결제분 환불 없음
- 결제 성공 시 즉시 상위 플랜 권한 적용
- 다음 정기결제일부터 상위 플랜 금액 청구
- anchor date 유지

## 5. 추가 계좌(add-on) 정책
- 중간 추가 시: 남은 기간에 대해 일할 계산 금액 즉시 청구
- 다음 정기결제일부터: 월 정상 금액(계좌당 3,000원) 합산 청구
- 최종 허용 계좌 수 = 플랜 기본 포함 수 + extra_account_slots

## 6. 결제 실패/유예
- 정기 결제 실패 시 유예기간 3일
- 유예기간 동안 기존 유료 권한 유지
- 유예기간 내 결제 성공 시 정상 복귀
- 3일 경과 후 실패 지속 시 past_due 전환 및 유료 권한 제한

## 7. 자동결제 및 고지
- 자동결제는 등록된 결제수단으로 청구됩니다.
- 결제 시도/실패/상태는 서비스 내 결제 상태 화면과 안내 문구를 통해 확인할 수 있습니다.

## 8. 저장 데이터 최소화 원칙
- 저장: orderId, paymentKey, 결제 상태/실패코드(정규화), subscription/item 스냅샷
- 암호화 저장: billingKey
- 저장 금지: raw 카드번호/CVC/민감 인증정보/authKey 장기 저장

## 9. 법률 검토 필요 사항
- 자동결제 사전고지 문구 및 동의 문구 최종 법적 형식
- 결제 실패 고지 주기/채널 고지 의무 범위
