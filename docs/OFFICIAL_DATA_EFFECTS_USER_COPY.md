# 공식 자료 반영 사용자 카피 가이드

## 핵심 톤
- 공식 자료 기준으로 보정했어요.
- 기준일이 있는 자료라서 언제 확인한 값인지 같이 보여드려요.
- 새 시즌이 오거나 숫자가 크게 달라질 때만 다시 확인할게요.

## 성공
- 이미 빠진 세금 기준값을 확인했어요.
- 공식 자료 기준으로 세금 숫자를 더 실제에 가깝게 맞췄어요.
- 건보료는 공식 자료 기준일과 납부 확인 상태를 함께 보여드려요.

## 일부 반영
- 일부는 공식 자료 기준으로 보정했고, 애매한 항목은 자동 반영하지 않았어요.
- 사업용 카드 사용내역은 자동 비용 확정이 아니라 참고 자료로만 유지해요.

## 재확인 권장
- 기준일이 지난 자료라 다시 확인하면 더 정확해져요.
- 자동 반영 대신 참고 상태로만 유지했어요.

## 확장 포인트 준비
- 공식 자료 기준으로 보정해 두면 다음 시즌에도 다시 확인하기 쉬워져요.
- 더 많은 항목을 자동 관리하려면 기준일이 최신인지 함께 보는 흐름이 필요해요.
