## 1-1. 국세청/홈택스 안내 확보 상태
- 2026-03-15 기준으로 법령정보센터 URL은 검증해 레지스트리에 포함했다.
- 국세청/홈택스의 설명형 안내 페이지는 공식 발행 주체와 현재 유효 URL을 함께 재검증한 뒤에만 추가한다.
- 따라서 이번 버전의 규칙 엔진 준비 문서는 `law.go.kr` 법령 링크를 source of truth로 사용한다.

## 2. 규칙 엔진에서의 사용 원칙

### 필요경비 일반 원칙
- 모든 비용처리 후보 판단의 출발점은 `사업 관련성`이다.
- 단순 업종 키워드 매칭만으로 비용처리를 확정하지 않는다.

### 개인·가사 관련 지출 제외
- 본인 식비, 생활비, 가정용 소비처럼 가사 관련 가능성이 높은 지출은 `do_not_auto_allow` 기본값으로 둔다.
- 개인/업무 혼합 가능 지출은 `needs_review` 또는 `do_not_auto_allow`로 보수적으로 내린다.

### 접대비·경조사비·선물
- 거래처 식사/접대비 후보는 참석자, 목적, 증빙 상태를 함께 확인해야 하므로 자동 확정 금지.
- 경조사비·선물은 일반 지출과 다른 검토 포인트가 있어 `consult_tax_review` 또는 최소 `needs_review`로 둔다.

### 적격증빙·증빙 수취
- 영수증이 있다고 바로 비용처리 확정으로 가지 않는다.
- 거래 사실, 증빙 유형, 사용자 메모, 매칭 거래 정보를 함께 본다.

## 3. 엔진 적용 시 금지 원칙
- 비공식 출처를 규칙 근거에 섞지 않는다.
- 법령상 불명확한 항목을 `high_likelihood`로 끌어올리지 않는다.
- 적격증빙이 없거나 사실관계가 약한 지출을 자동 인정하지 않는다.

## 4. 추가 검토 메모
- 이번 레지스트리는 2026-03-15 기준으로 URL과 조문 기준을 고정했다.
- 향후 국세청/홈택스 공식 설명 페이지를 추가할 때도, URL과 발행 주체가 공식인지 먼저 검증한 뒤 포함한다.
