# SafeToSpend 소개서 (비개발자용)

작성일: 2026-03-08

## 1) 이 서비스는 한마디로?
SafeToSpend는 프리랜서/1인 사업자가  
"이번 달 돈 정리와 세금 준비를 늦지 않게 끝내도록" 돕는 서비스입니다.

## 2) 왜 필요한가?
많은 사용자가 월말에 아래 문제를 겪습니다.
- 거래는 많은데 수입/지출 구분이 아직 안 됨
- 영수증이 빠져 있어 세무사 전달 전에 다시 찾게 됨
- 세금/건보료를 마지막에 알게 되어 현금이 부족해짐

SafeToSpend는 이 문제를 "월중에 미리" 정리하도록 설계돼 있습니다.

## 3) 사용자가 실제로 받는 가치
- 오늘 당장 해야 할 일 1개를 보여줌
- 누락된 영수증을 먼저 잡아줌
- 이번 달에 미리 보관해야 할 세금/건보료(추정)를 알려줌
- 세무사에게 전달할 파일을 패키지로 묶어줌

## 4) 이 서비스가 하는 일 (쉬운 흐름)
1. 거래 파일을 가져옵니다.
2. 서비스가 수입/지출/증빙 상태를 정리해 보여줍니다.
3. 사용자는 "필수 누락"부터 빠르게 처리합니다.
4. 서비스가 세금/건보료(추정)를 기준으로 보관 가이드를 줍니다.
5. 마지막에 세무사 전달용 패키지를 내려받습니다.

## 5) 주요 화면은 이렇게 보면 됩니다
- `/overview`: 이번 달 한눈에, 지금 할 일
- `/dashboard/calendar`: 거래 달력과 월간 흐름
- `/dashboard/review`: 분류/증빙 정리의 중심 화면
- `/dashboard/tax-buffer`: 세금 보관금 점검/조정
- `/dashboard/package`: 세무사 전달용 패키지 생성
- `/dashboard/nhis`: 건보료 추정 확인과 입력 보완

## 6) "건보료 화면"은 정확히 뭐냐?
건보료 화면은 "월 건보료(추정)를 미리 보는 화면"입니다.
- 현재 월 기준 금액
- 11월 반영 기준 금액
- 왜 이렇게 나왔는지(소득/자산 영향)
- 입력을 더하면 정확도가 올라감

중요:
- 화면의 숫자는 안내용 "추정"입니다.
- 확정 고지서 금액과는 차이가 날 수 있습니다.

## 7) 사용자 데이터는 어떻게 다루나?
- 입력값은 계산/정리에 필요한 범위에서 저장됩니다.
- 증빙은 보관함과 패키지 생성 흐름에서 사용됩니다.
- 관리자 화면과 사용자 화면은 분리되어 운영됩니다.

## 8) 이 서비스가 약속하는 기준
- 복잡한 월말 작업을 순서대로 줄여준다
- 사용자가 "지금 무엇을 해야 하는지" 먼저 보여준다
- 추정 숫자는 과신하지 않도록 표시한다
- 실패해도 가능한 한 화면이 깨지지 않게 동작한다

## 9) 이런 분에게 특히 맞습니다
- 매달 세무 정리가 밀리는 프리랜서
- 매출은 있는데 세금/건보 준비가 불안한 사용자
- 세무사 전달 전에 자료 품질을 스스로 점검하고 싶은 사용자

## 10) 한 줄 결론
SafeToSpend는 "기록 앱"이 아니라  
"월말 리스크를 줄이는 운영 보조 서비스"입니다.
