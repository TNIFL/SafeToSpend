# 공식 자료 source of truth 가이드

## 목적
- 쓸수있어에서 공식 자료 업로드 UX, 카피, 바로가기, 메뉴 경로 안내를 만들 때 기준으로 삼는 문서다.
- 제품 카피와 화면 안내는 이 문서의 자료명, 공식 사이트, 메뉴 경로를 우선 기준으로 쓴다.

## source of truth 원칙
- 공식 사이트는 `hometax.go.kr`, `nts.go.kr`, `nhis.or.kr`, `law.go.kr`만 사용한다.
- 바로가기 링크가 실패하더라도 사용자가 스스로 찾을 수 있도록 메뉴 경로 텍스트를 함께 보여준다.
- 홈택스 메뉴명은 개편에 따라 일부 달라질 수 있으므로, 화면에는 "바로가기 + 실패 시 메뉴 경로"를 같이 둔다.

## 공식 바로가기와 실패 시 경로
### 홈택스
- 공식 사이트: `https://www.hometax.go.kr/`
- 보조 공식 안내: `https://mob.hometax.go.kr/home/mtmhighmtmhigh100M01.do?metaTy=MANUAL`
- 사업용 카드 공식 상담/도움말 예시: `https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1311&ctgId=CTG11924&ctgSubId=CTGS12158`
- 링크가 안 열리면 먼저 찾을 묶음 메뉴:
  - `전자(세금)계산서·현금영수증·신용카드`
  - `조회/발급`
  - `My홈택스`

### NHIS
- 공식 사이트: `https://www.nhis.or.kr/`
- 공식 민원안내: `https://www.nhis.or.kr/nhis/minwon/minwonGuide.do?mode=view&articleNo=146671`
- 링크가 안 열리면 먼저 찾을 메뉴:
  - `민원서비스`
  - `서비스찾기`
  - `증명서 발급/확인`

## 자료별 쉬운 설명
### 현금영수증 지출증빙 내역
- 사용자에게 보여줄 이름: `현금영수증 지출증빙 내역`
- 왜 필요한가: 누락된 지출증빙이 있는지 확인해 비용 반영 정확도를 올린다.

### 사업용 카드 사용내역
- 사용자에게 보여줄 이름: `사업용 카드 사용내역`
- 왜 필요한가: 카드로 쓴 비용을 한 번에 맞춰서 review 수작업을 줄인다.

### 전자세금계산서 내역
- 사용자에게 보여줄 이름: `전자세금계산서 내역`
- 왜 필요한가: 매입·매출 증빙을 세무사 전달 자료와 맞추는 데 도움을 준다.

### 원천징수·지급명세·납부내역
- 사용자에게 보여줄 이름: `이미 빠진 세금/지급명세 자료`
- 왜 필요한가: 세금 정확도와 이미 빠진 세금 반영 상태를 더 정확히 본다.

### 보험료 납부확인서
- 사용자에게 보여줄 이름: `건보료 납부확인서`
- 왜 필요한가: 추정 건보료와 실제 납부 흐름 차이를 줄인다.

### 자격득실확인서
- 사용자에게 보여줄 이름: `건보 자격 변동 확인서`
- 왜 필요한가: 직장가입자·지역가입자·피부양자 전환 시점을 설명하는 데 쓴다.
